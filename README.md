# Rock-Paper-Scissors with AI

A simple console-based Rock-Paper-Scissors game in Python with an AI opponent.  
The AI predicts your moves based on your play history.

## How to Play
- Type 0 for Rock
- Type 1 for Paper
- Type 2 for Scissor
- Type -1 to quit
## How to Run
Run the game with:
   ```bash
   python main.py
